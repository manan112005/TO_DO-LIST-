# 📝 To-Do List Web App

A simple and interactive To-Do List web application that helps users manage daily tasks efficiently.  
All tasks are saved in the browser, so they remain even after refreshing the page.

## 🚀 Features
- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 💾 Tasks are saved automatically (no reset on refresh)
- 📱 Works on mobile and desktop browsers

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 💡 How It Works
- Tasks are stored in the browser using Local Storage
- No internet or backend required
